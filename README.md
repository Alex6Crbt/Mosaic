## Getting Started



### Prerequisites

Make sure you have Node and npm installed

### Installation

To run locally the app run the folowing :

1. Clone the repo
```sh
git clone https://github.com/alex6crbt/Mosaic.git
```
3. Install NPM packages
```sh
npm install
```
4. Start the app
```sh
npm run dev
```
